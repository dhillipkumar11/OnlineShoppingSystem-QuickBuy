# QuickBuy - Online Shopping System

QuickBuy is a simple online shopping system built using **Spring Boot**.  
It allows users to browse products, add them to cart, and place orders.

## Features
- Add to Cart
- View Cart
- Place Order
- Product Management

## Tech Stack
- Java
- Spring Boot
- Maven
- HTML, CSS
- MySQL (for database)
